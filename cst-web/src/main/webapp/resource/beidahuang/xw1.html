黑龙江省完成62项绿色食品地方标准制修订工作

来源：黑龙江省省绿色发展中心

添加时间：2017年02月17日

绿色食品标准是绿色食品产业发展的重要技术基础，是科技成果转化为现实生产力的桥梁和纽带。及时进行绿色食品地方标准的制修订是企业、基地和农户实施标准化生产、提高产品质量和满足相关产品认证要求的重要保障，对绿色食品产业的持续健康发展具有重要意义。

为进一步做好地方标准的制修订工作，黑龙江省绿色食品发展中心2016年共制修订了62项绿色食品地方标准，其中，修订标准61项，制定标准1项。省绿色食品发展中心在专家库中组织了71名相关领域的专家，耗时近两个月，完成了标准的修订和征求意见工作，省质监局又组织86名相关专家开展了标准评审工作，62项标准于11月3日由省质监局2016年第42号公告发布实施。

及时制定、颁布和实施绿色食品地方标准，将有效地推动绿色食品生产向科学化、规范化和制度化方向发展，维护我省绿色食品质量和信誉，提高我省绿色食品市场占有率，促进农业大省向绿色食品大省转变。
