前哨100%秸秆还田发展绿色农业

来源:北大荒日报

添加时间：2016年9月13日

近日，笔者从前哨农场相关部门了解到，这个农场171台大中型联合收割机、1263台小履带收割机、31台拖拉机配套的秸秆粉碎还田机已经全部检修、调试完毕。农场还要求今年新购置的各类收割机和外引跨区作业收割机必须配有秸秆粉碎器，否则不允许参加作业。这是这个农场积极推广秸秆还田技术，大力发展绿色农业的一个缩影。

为实现全场水田100%秸秆还田，今年，这个农场加大了对秸秆粉碎还田技术的宣传力度，要求各农场组织农业技术人员深入田间地头，向农户宣传讲解秸秆还田在增加土壤中有机质含量、培肥地力、改善土壤结构、提高农产品品质等方面的优势，进一步提高农户对秸秆还田的认识。

同时，在秋收机具检修过程中，这个农场要求有机户全部安装秸秆粉碎器，并指导他们了解、掌握秸秆粉碎器的操作技术和规程。为保证秸秆还田技术的推广和应用，农场还将秸秆还田工作纳入到秋收方案中，专门成立了领导小组，结合农场实际，出台了秸秆还田机械补贴政策，鼓励农户购买、使用秸秆还田机具，提高秸秆还田率。

此外，在秋收期间，这个农场还将成立由环保、农业、农机等部门组成的秸秆禁烧工作督查组，值守在秋收现场，抓好秸秆还田工作落实，确保秸秆还田全覆盖、无盲区。
