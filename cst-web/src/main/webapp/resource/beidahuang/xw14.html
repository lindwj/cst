北大荒金秋遍地是“黄金”

来源：黑龙江省省绿色发展中心

添加时间：2016年09月22日

北大荒的原野热闹非凡，成片的大豆摇动着豆荚，发出了哗啦啦的笑声；挺拔的高粱扬起黑红黑红的脸庞，象是在乐呵呵地演唱。山坡上，大路边，村子口，榛树叶子全都红了，红得像一团团火，把人们的心也给燃烧起来了。生活在这里的人，喜欢用“大豆摇铃千里金”来赞美遍地金黄的九月。这里的田野、山岭、江河，连同那茫茫的草甸子，都会在这个季节里用双手捧出沉甸甸的宝物来。我们北大荒的农场也是收获满满！

建设农场的红高粱长势喜人

今年，建设农场加大科技品种更新投入力度，以科技园区为纽带推动农场品种更新，调整优化全场种植结构，扩大绿色经济作物种植面积。目前，科技园经济作物面积1.15万亩，种植了红高粱、花芸豆、红小豆等各类经济作物，为全场大面积品种推广奠定了基础。图为农场综合服务科技园区农业技术人员正在预估即将收获的“绥杂8号”高粱产量。

汤原农场种植金花葵拓宽致富路

连日来，汤原农场金花葵种植户积极组织工人进行采收、烘干、装袋，并进行网上销售。据了解，金花葵，又名野芙蓉，为一年生草本锦葵秋葵属植物，2003年8月被中国农科院在河北邢台地区发现濒临绝种的植物。在200多个球葵植物中最具食用药用保健功能，被誉为植物届大熊猫，营养成份比金秋葵高1.6倍，有较高的利用价值。今年，这个农场居民种植了10亩金花葵，第一批留籽，现正在进行采摘作业。

龙门农场首次发现野生大豆

近日，龙门农场科技人员在第四管理区境内发现多株野生大豆。野生大豆属一年生草本植物，根茎细长具有藤缠性特点，一般依附其它植物生长，也可匍匐性生长，别名豆、乌豆、野黄豆等，是国家二级保护野生植物，通过对野生大豆习性的研究，可帮助培育寒地大豆新品种提供参考依据，因此具有很高的科研价值。

兴凯湖平欧大榛子拓宽致富路

今年，北大荒股份兴凯湖分公司员工刘振国利用兴凯湖湖岗培育种植的40亩平欧大榛子果实已挂满枝头。据了解，刘振国种植的平欧大榛子平均亩量达到300斤，价格为20元每斤，每亩收入可达到6000元以上，经济效益十分可观。

八五二“富硒”木耳喜获丰收

今年，北大荒股份八五二分公司2980栋水稻育秧大棚全部实现二次利用，除种植瓜果类、蔬菜类、高效经济作物类等农作物，还栽培食用菌。其中“富硒”黑木耳，产量高、营养价值丰富，生长采摘期长达四个月，每袋采摘收获晒干的木耳可达0.1斤。这种黑木耳中硒的含量是一般木耳的2~3倍，有预防高血压、高血脂、高血糖“三高”功效，深受消费者欢迎。图为这个分公司第五管理区高标准水稻育秧大棚内，“富硒”木耳种植户正在高兴地展示丰收的“富硒”木耳。

勤得利葡萄串串迎金秋

北大荒股份勤得利分公司葡萄种植基地里熟透了的葡萄、提子挂满枝头，煞是喜人。种植户忙着采摘葡萄、提子到市场上出售，赶在节日期间卖个好价钱，增加收入。

勤得利小猴头带来大效益

今年北大荒股份勤得利分公司的气候条件，非常适宜山上的猴头菌生长。员工们三三两两提筐背篓进山采摘猴头，每公斤猴头在市场销售价格为100元，每天每人可采摘猴头5公斤。

前进农场大球盖菇试种成功

前进农场科技园区，一朵朵红色大蘑菇破土而出，这标志着这个农场试种大球盖菇获得成功。

这个农场推进水稻大棚二次利用，在种植红小豆、玉米、刺五加、木耳等经济作物的同时，又研究试验经济价值高、市场前景好的大盖菇养殖。今年4月水稻秧苗下地后，科技园区就用水稻秸杆、稻壳、牛粪等做培养基进行发酵，经过几个月的辛苦试验获得成功。

大球盖菇是近几年刚刚兴起的食用菌新秀，富含蛋白质和多种矿物质元素及维生素，是集香菇、蘑菇、草菇三者于一身的美味食品，具有“素中之荤”的全价营养保健食品，大球盖菇还具有预防冠心病、助消化、疏解人体精神疲劳之功效，市场前景非常好。
